PHP Web Application Project

Leave Management System

A clean, efficient, and simple web application developed in PHP and MySQL to manage employee leave records, requests, and approvals seamlessly.

Get Started / Run Locally View GitHub Code

Project Structure & Modules

Fully functional modules designed for role-based workflows

Client / Employee Console

Designed for employees to manage leaves, track profiles, and request time off.

  • Submit Leave Requests easily
  • View remaining balances (Casual, Sick, Earn)
  • Upload and change profile pictures
  • Secure password recovery via email key

Admin Management Console

Enables company/organization administrators to control employee profiles and leaves.

  • Grant or Reject pending leave requests
  • Set and update general leave policies
  • Search, filter, or delete employees
  • Export leave logs directly as PDF reports

Automated Notifications

Integrates third-party APIs to automate email alerts for actions.

  • PEAR Mail SMTP configuration ready
  • Email registration confirmations
  • Leave request grant/reject status notifications
  • DOMPDF library for report downloads

Installation & Local Setup Guide

01

Clone the Repository

Download the project using Git to your local directory (e.g. `htdocs` in XAMPP).

git clone https://github.com/vijaymahes9080/Leave-Management-System-Project-PHP.git
02

Import Database

Launch phpMyAdmin, create a database named phpdb, and import the database file:

phpdb.sql
03

Configure Mailer

In `client/recoverymailer.php`, `client/leave_mailer.php`, `admin/register/mailer.php`, and `admin/mailer.php`, configure your SMTP credentials:

'username' => 'your_email@gmail.com',
'password' => 'your_app_password'
